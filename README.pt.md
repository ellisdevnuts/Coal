<br>

Coal é um app de chat desktop feito pra comunidades brasileiras que querem algo leve, honesto e seu. Sem anúncios. Sem verificação de idade. Sem enrolação corporativa. Só servidores, canais e pessoas.

A inspiração vem do que o Stoat se propôs a ser — aberto, movido pela comunidade, livre do peso que o Discord foi acumulando ao longo dos anos — mas construído do zero com uma stack mais simples e focado em não atrapalhar ninguém.

<br>

## o que é

Servidores, canais, mensagens em tempo real, upload de arquivos, perfis, convites — o que importa. Uma interface escura com cores e fontes personalizáveis, porque você passa bastante tempo aqui e isso deveria parecer certo.

Roda como app desktop no Linux e Windows via Electron, e usa o Supabase por baixo para autenticação, banco de dados e comunicação em tempo real.

<br>

## limitações

O Coal roda no plano gratuito do Supabase, o que significa que existem limites reais.

500MB de armazenamento total no banco. 50.000 usuários ativos por mês. Sem canais de voz ou vídeo. Sem app mobile. Uploads de arquivos passam pelo Catbox, um host externo, então arquivos podem sumir se o Catbox os remover. Sem criptografia de ponta a ponta.

Essas não são coisas pra esconder. Se o projeto crescer, a infraestrutura vai precisar crescer junto.
